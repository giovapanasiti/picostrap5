Hello, picosass!

This page compiles SCSS code in the browser. In detail, it compiles the full Bootstrap 5.3.1 CSS library, reading SCSS files hosted on the CDN.

Of course, this is not for production use. You should always "staticize" your CSS bundle.

We built this module for Picostrap, a starter theme implementing SASS and Bootstrap for WordPress.

How to compile SCSS in the browser on my own web page / app

View this page source code. It's really easy!

Just add the < script> import tag, and some < template id = "the-scss" > element containing the SCSS to be compiled.
